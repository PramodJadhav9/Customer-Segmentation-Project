# Customer Segmentation Project

## Objective

To segment customers into different groups based on their annual income and spending behavior using the K-Means clustering algorithm.

## Tools Used

* Python
* Pandas
* Matplotlib
* Scikit-learn

## Dataset

The project generates data for 100 customers with:

* Age
* Annual Income
* Spending Score

## Method

K-Means clustering with 3 clusters was applied to group customers with similar characteristics.

## Output

* customer_data.csv (dataset)
* customer_segments.png (visualization)
* customer_segmentation.py (source code)

## Conclusion

Customer segmentation helps businesses understand different customer groups and create targeted marketing strategies.
